# TP 01 - Premier boot et terminal

Objectif : identifier la machine et comprendre les premieres commandes.

## Commandes a executer

```bash
whoami
hostname
pwd
uname -a
lsb_release -a
ip address
df -h
free -h
```

## Questions

- Quel est votre utilisateur ?
- Quel est le nom de la machine ?
- Quelle est la version du kernel ?
- Quelle est la distribution ?
- Quelle est l'adresse IP principale ?
- Combien de RAM est disponible ?

## Aide locale

Trouver l'option qui affiche les tailles lisibles par humain.

```bash
man ls
ls --help
ls -lh
```

## Livrable

Creer :

```bash
mkdir -p ~/linux-lab/notes
nano ~/linux-lab/notes/system-notes.md
```

Le fichier doit contenir les reponses aux questions.
